Our tool is to be used under python3 exclusively.

To run de tool just type "./tool slice.json pattern.json" or "python3 slice.json pattern.json"
under src directory. The output file will be written under the directory of the input file. 

We also have a tool to run our own set of tests (these set includes the ones given in course web page). 
You can try it by typing "python3 godzila.py" or "./godzila"
